# vue快速启动

1. 先安装vue `npm create vue@latest`

2. 然后拷贝原代码放入新项目目录，安装需要的库

```
npm install bootstrap

npm install sass

npm i bootstrap-icons

npm i @kangc/v-md-editor@next -S

npm install chart.js

npm i -D naivu-ui

npm install axios
```

最后进入项目执行启动
